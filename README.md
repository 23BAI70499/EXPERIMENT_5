# Unit-5: Optimize Frontend Performance Using Lazy Loading

This project demonstrates how to optimize frontend performance in a React Single Page Application (SPA) using **lazy loading** with **React.lazy()** and **Suspense**, including **route-based lazy loading** using **React Router DOM**.
