# React Router + Lazy Loading

Simple Vite + React project using:
- React Router (`/dashboard`, `/profile`)
- Lazy Loading with `React.lazy()`
- Loader using `Suspense`

---
Route-based lazy loading means:
- Components/pages are loaded only when the user visits that route.
- It reduces initial bundle size.
- It improves loading speed of the application.

---

## Technologies Used
- React.js
- React Router DOM
- CSS
